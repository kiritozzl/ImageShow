# ImageShow
简单的图库，获取SD卡里的图片，主界面有list和grid两种显示模式；


在Grid模式可以批量删除图片，主界面均可以从menu调用系统摄像头，拍照，并保持所拍照片；


显示一张图片时可以删除图片，调用其他应用分享图片，双指缩放图片，单指移动图片，三指选择图片；


一张图片的删除效果不太好，有待优化，显示大图片时的outofmemory的问题依然没有解决！
